IRC Server

Mar 2025 - May 2025

Multi-client IRC server built from scratch in C++ as part of 42 curriculum. Collaborated in 3-person team to implement network programming fundamentals, concurrent client handling, and IRC protocol compliance using object-oriented design patterns.

Technical Implementation

  • • Network Programming: Socket programming with non-blocking I/O and select()
  • • Protocol: IRC RFC compliance for commands, channels, and private messaging
  • • Concurrency: Multi-client handling with single-threaded event loop
  • • Design Patterns: Command pattern for IRC command processing
  • • OOP Principles: Encapsulation, inheritance, and polymorphism in C++
  • • Memory Management: Manual resource management without memory leaks

Key Features: Channel creation and management, user authentication, operator privileges, private messaging, channel modes (invite-only, topic restrictions, passwords), kick/ban functionality, and proper error handling. Compatible with standard IRC clients like irssi and WeeChat.
