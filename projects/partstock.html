Auto Parts Manager

Sep 2025 - Nov 2025

Full-stack inventory management system for used auto parts business. Enabled 50% sales growth in first month through automated OLX marketplace integration. Features role-based authentication, audit logging, and image processing pipeline deployed on AWS infrastructure.

Key Features

  • • FastAPI REST API backend
  • • SQLAlchemy ORM with SQLite
  • • Role-based access control (RBAC)
  • • Complete audit trail logging
  • • Automated OLX marketplace sync
  • • Image upload and processing
  • • AWS EC2 + S3 deployment
  • • Docker containerization

Business Impact: Automated marketplace integration reduced manual posting time by 80%, allowing focus on customer service and inventory acquisition. System handles 200+ part listings with full inventory tracking and sales analytics.
