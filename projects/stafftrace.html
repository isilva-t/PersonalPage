StaffTrace Monitor

Nov 2025 - Present

Microservices attendance tracking system with dual architecture: on-premise monitors for network device detection and cloud dashboard for multi-site data aggregation. Demonstrates full-stack deployment capability from local infrastructure to cloud orchestration.

Cloud Dashboard

  • • Angular frontend with TypeScript
  • • Java Spring Boot REST API
  • • MongoDB for multi-site aggregation
  • • Kubernetes (GKE) orchestration
  • • GitHub Actions CI/CD pipeline

On-Premise Monitor

  • • Python Django backend
  • • PostgreSQL local database
  • • Redis for caching and sync queue
  • • Network device detection
  • • Automated cloud synchronization
