3D Ray Tracer

Nov 2024 - Jan 2025

Ray tracing renderer built in C as part of 42 Porto curriculum. Implemented from scratch using Test-Driven Development to validate complex mathematical operations before implementing rendering features. Demonstrates solid understanding of computer graphics, linear algebra, and low-level programming.

Technical Implementation

  • • Ray Tracing Algorithm: Light ray simulation with reflection, refraction, and shadows
  • • 3D Primitives: Spheres, planes, cylinders, and cones with proper intersection calculations
  • • Matrix Operations: Camera transformations, rotations, and translations
  • • Lighting Models: Ambient, diffuse, and specular illumination (Phong reflection model)
  • • TDD Approach: Unit tests for mathematical operations before feature implementation
  • • Performance: Optimized intersection tests and efficient memory management

Key Learning: This project strengthened understanding of vector mathematics, 3D geometry, and algorithm optimization. The TDD approach taught the value of validating complex mathematical operations incrementally before building higher-level features.
